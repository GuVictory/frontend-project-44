## How to play Brain Games

1. Clone repo:
  ```bash
    git clone https://github.com/GuVictory/frontend-project-44.git
  ```

2. Publish npm package with `--dry-run`
  ```bash
    make publish
  ```

3. Link npm package
  ```bash
    npm link
  ```

4. Start any of avelible games
